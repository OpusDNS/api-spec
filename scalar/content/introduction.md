# Introduction

Welcome to the **OpusDNS Developer Documentation**.

This site brings together everything you need to integrate with the OpusDNS
API, including:

- **Guides** — task-oriented walkthroughs (Quickstart, Authentication, …).
- **API Reference** — the full OpenAPI specification, generated from
  [`src/openapi.yaml`](https://github.com/OpusDNS/api-spec/blob/main/src/openapi.yaml)
  in this repository.
